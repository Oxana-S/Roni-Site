# Состояние проекта — Roni Site

**Обновлено:** 2026-06-28
**Сайт:** писатель Ростислав Вишневский (эпическое фэнтези). Статический, движок dc-runtime.

## Где живёт
- **Репозиторий:** https://github.com/Oxana-S/Roni-Site (ветка `main`, публичный)
- **Превью (текущее):** https://oxana-s.github.io/Roni-Site/ (GitHub Pages)
- **Прод (план):** свой сервер + домен — **домена пока нет**. Сайт уже server-ready
  (относительные пути). Деплой — `docs/DEPLOY.md`.

## Стек
- Статический HTML/CSS/JS. Движок **dc-runtime** (`support.js`): компоненты `*.dc.html`,
  `{{ }}`, `<sc-for>`, `<dc-import>`, `class Component extends DCLogic` + `renderVals()`.
- React 18.3.1 + Babel 7.26.4 — **локально** (`assets/vendor/`, без CDN), с SRI.
- Многоязычность RU/EN/UK/BG (клиентская, `localStorage('vish_lang')`).
- Дизайн-токены: `styles/theme.css` (единый источник). Живой каталог: `design-tokens.html`.

## ✅ Что сделано (всё на проде)
- **Латинские URL** для всех 16 страниц (кириллицы в адресах нет); язык запоминается.
- **Дизайн-система:** токены цвета/типографики/spacing/ширин в `theme.css`; золото — `--gold-rgb`.
- **Шапка:** логотип (гексагон) · меню по центру · языки справа · бургер на мобиле.
- **Секции главной** центрированы; «голос автора» — фото справа.
- **Размер текста** для слабовидящих (3 уровня) — в футере, запоминается.
- **Доступность:** `<html lang>`, aria в шапке, focus-visible, reduced-motion, тач-таргеты.
- **SEO:** title/description/keywords/canonical/OG/Twitter на всех страницах; внутренние —
  `noindex`. Фавиконки, `sitemap.xml`, `robots.txt`, `site.webmanifest`, `og-image`.
  **Многоязычные мета** (`assets/seo-meta.js`) — меняются по языку.
- **Реальные ассеты:** логотип, портрет автора, обложка книги 1, hero-карта мира.
- **Форма контактов:** вводимый текст читаемый (был тёмный на тёмном).
- **Аудит «API»** (4 ревьюера) + чистка: мёртвый код шапки убран, 404/dev на компоненте,
  `document.lang` в lifecycle. Дев-документация — `docs/API.md`.

## ⏳ Что осталось / нужно от пользователя

### Нужны файлы от пользователя
- **Аудио** `assets/audio/chapter-1.mp3` — запись первой главы (плеер на главной ждёт её;
  сейчас при Play будет 404).
- **Картинки-заглушки** (размеры в `docs/IMAGES-spec.md`): `book-2.jpg` (вторая книга),
  `world-earth.jpg` / `world-tolus.jpg` (секция «Два мира»), `blog-1.jpg`…`blog-6.jpg` (блог).

### Когда появятся домен и аналитика
- **Домен:** `python3 scripts/set-domain.py https://домен.com` → загрузить на сервер.
- **Google Analytics:** вписать `GA_ID` в `assets/analytics.js` (одна строка).
  ⚠️ GDPR (ЕС): GA с cookie требует согласия — при необходимости сделать баннер.

### Минорный бэклог (косметика, не срочно) — `docs/API.md` §10
- alt-тексты картинок на RU → в `{{ t.* }}`; ~9 разных URL Google Fonts → унифицировать;
  остаток палитры (`#a89c84` и др.) вне токенов; `directions.html`/`wireframe.html` — мокапы.

### Не проверено
- **Полноценный мобильный QA** — адаптив заложен в коде, но попиксельно на телефоне не
  выверен (инструмент скриншотов не отражает мобильную ширину). Глянуть на реальном устройстве.
